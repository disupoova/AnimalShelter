Об авторе



Данное приложение для приюта животных "Краснокот" для Курсовой работы по дисциплине "Современные технологии программирования" выполнила Мерзлова Анастасия Юрьевна, студентка Финансового Университета При Правительстве Российской Федерации, обучающаяся в группе ПИ21-2 по программе "Прикладная информатика" на 2 курсе.

Веб-приложение выполнено на языке программирования Java с использованием технологий Spring Boot, Thymeleaf, MySQL, ORM, Spring Security. Реализована веб-служба Restful API.

Референсом для данного приложения является уже существующий приют.

Автор выражает благодарность приюту для животных "Краснодог" в городе Краснодар за их работу и благотворительную деятельность. Все представленные реквизиты принадлежат им.

Их контакты: Сайт приюта "Краснодог"
